# App de escritorio para Peluquería Canina

App desarrollada con Java Swing con conexión a base de datos MySQL.

## Tecnologías

- Java JDK 17
- Java JPA
- Maven

## Dependencias

- MySQL Driver
- Hibernate
- Lombok
